1、查看磁盘和分区状态
```bash
lsblk
```
在输出的结果中，你应该能看到一块主硬盘（通常叫 sda 或者如果是 NVMe 的话叫 nvme0n1）。它的总容量应该已经变成了你扩容后的大小，但是它下面的子分区（比如 sda2 或 sda3）还是原来的大小。记下那个你需要扩容的主分区编号。

2、安装必备工具
在终端输入以下命令，更新一下软件列表并安装包含 growpart 的工具包：

```bash
sudo apt update
sudo apt install cloud-guest-utils -y
```

3、重新执行扩容分区命令
安装完成后，再次执行刚才那条命令把“围墙”拉伸：

```bash
sudo growpart /dev/sda 3
```
顺利的话，这次终端会输出一行包含 CHANGED 字样的信息，代表底层分区拉伸成功。

3、扩容文件系统
最后一步，通知系统文件管理器空间变大了：

```bash
sudo resize2fs /dev/sda3
```
这三步跑完之后，你再输入 df -h / 看一眼，新增的 10G 空间就实打实地全部可用了！
